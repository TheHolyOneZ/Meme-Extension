# Meme.py

**meme.py** is a fun tool that generates random memes from various sources. You can use it to bring a little humor to your day by simply running a few commands. Here's everything you need to know!

## Commands

You can use the following commands to interact with **meme.py**:

- `!meme`: This command will generate a random meme from the available sources.
- `meme_help`: Use this command to get more information about the available commands and their usage.
- `!meme_list`: This command will display the list of sources from which memes can be generated.

## Meme Sources

**meme.py** fetches memes from the following sources:

1. [https://meme-api.com/gimme](https://meme-api.com/gimme): General random memes.
2. [https://meme-api.com/gimme/dankmemes](https://meme-api.com/gimme/dankmemes): A collection of dank memes.
3. [https://meme-api.com/gimme/funny](https://meme-api.com/gimme/funny): Memes that are typically humorous or light-hearted.
4. [https://meme-api.com/gimme/ProgrammerHumor](https://meme-api.com/gimme/ProgrammerHumor): A selection of memes aimed at programmers.
5. [https://meme-api.com/gimme/memes](https://meme-api.com/gimme/memes): A variety of memes to suit any taste.

## How to Use

1. Install **meme.py** and its dependencies.
2. Use the commands mentioned above to fetch memes.
3. Enjoy the humor!
